# SPEC-001 — Registro de decisiones

## DEC-001 — React con Vite

**Estado:** Aceptada. No se necesita SSR ni backend Node; Vite reduce complejidad.

## DEC-002 — SQLite

**Estado:** Aceptada. Favorece local-first, instalación mínima y migración futura mediante SQLAlchemy.

## DEC-003 — Factory de FastAPI

**Estado:** Aceptada. Facilita inyección de configuración y pruebas aisladas.

## DEC-004 — OpenAPI como contrato

**Estado:** Aceptada. El contrato backend-frontend se derivará de OpenAPI.

## DEC-005 — No crear dominio ficticio

**Estado:** Aceptada. SPEC-001 solo crea capas; las entidades lingüísticas llegarán con requisitos funcionales.

## DEC-006 — Docker opcional

**Estado:** Aceptada. Los comandos nativos son la ruta principal; Docker Compose es apoyo de reproducibilidad.
